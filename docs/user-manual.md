# AI 应用广场用户操作手册

## 1. 手册说明

- 适用系统：河北 AI 应用广场（当前版本）
- 适用范围：匿名浏览、登录申报、管理员审核与管理
- 阅读建议：先看“角色导航”，再按自己的角色执行对应步骤
- 特别说明：本手册以当前系统实际功能为准，不包含未来外部 SSO/OA 对接方案

## 2. 角色导航

- 匿名用户：可浏览首页榜单、应用视图、应用详情、历史榜单
- 登录用户：可在浏览基础上进行“我要申报”“我的申报”
- 管理员：可在登录基础上访问管理端（申报审核、榜单管理、用户管理）

## 3. 匿名用户操作路径

### 3.1 浏览首页

1. 打开系统首页。
2. 首页默认展示排行榜，可查看“总应用榜”和“增长趋势榜”。
3. 点击“应用视图”查看集团应用和省内应用展示内容。
4. 使用应用来源、状态、分类、公司、关键词等筛选条件定位应用。

## 4. 登录用户操作路径

### 4.1 登录

1. 在首页点击“我要申报”或“登录”入口。
2. 进入登录页后输入账号密码并提交。
3. 如账号使用初始密码或管理员重置密码，系统会先要求修改为个人强口令。
4. 改密完成后返回业务页面。

登录提示口径：
- 登录后可提交申报，未登录仅可浏览。
- 强口令至少 10 位，且需包含大写字母、小写字母、数字、符号中的至少三类。

### 4.2 我要申报（核心流程）

1. 未登录时点击“我要申报”，系统跳转登录。
2. 如账号需要修改初始密码，先完成改密。
3. 登录与必要改密完成后自动回到首页，并自动打开“申报弹窗”。
4. 按要求填写基础信息、应用信息、成效信息。
5. 上传封面图和详细文档（如有）。
6. 点击“提交申报”完成提交。

1. 登录后点击“我的申报”进入列表。
2. 查看本人申报记录与审核状态。
3. 对 `pending`（待审核）记录可执行修改或撤回。

### 4.4 主动修改密码

1. 登录后在首页右上角点击“修改密码”。
2. 输入当前密码、新密码和确认密码。
3. 新密码需满足强口令规则：至少 10 位，且需包含大写字母、小写字母、数字、符号中的至少三类。
4. 修改成功后返回首页，当前会话继续有效。

### 4.5 状态说明

- `pending`：待审核，可修改/撤回
- `approved`：审核通过，已转应用
- `rejected`：审核未通过，可按反馈重新整理后再申报
- `withdrawn`：已撤回，不再进入当前审核

## 5. 管理员操作路径

### 5.1 管理员登录入口

1. 在未登录首页点击“管理员登录”。
2. 使用管理员账号登录。
3. 登录后进入管理功能页面。

说明：
- 普通登录用户不可访问管理页，系统会提示无权限并返回首页。

3. 新建或重置用户密码时必须填写强口令；该密码视为临时密码，目标用户下次登录后需要先修改。
4. 保存后验证目标用户访问权限是否符合预期。

## 6. 常见问题（FAQ）

### 6.1 登录失败怎么办？

- 先确认账号密码是否正确
- 若连续失败，请联系管理员重置账号状态

### 6.2 为什么我看不到“我的申报”？

- 未登录状态不会显示“我的申报”
- 请先登录，再回到首页查看顶部入口

### 6.3 为什么访问管理页被拒绝？

- 当前账号不是管理员角色
- 请联系管理员确认账号角色配置

### 6.4 申报提交失败怎么办？

- 先检查必填项是否完整
- 检查图片/文档格式与大小是否符合要求
- 重试后仍失败，记录时间与操作步骤反馈管理员排查

### 6.5 页面打不开或样式异常怎么办？

- 刷新浏览器并重新登录
- 若系统部署在 `/AISquare/` 子路径，请确认访问地址包含该前缀
