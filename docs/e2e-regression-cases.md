# E2E 回归用例（Batch-2）

## E2E-001 申报封面图可移除
1. 进入首页，点击“我要申报”打开申报弹窗。
2. 上传一张 JPG/PNG 封面图，确认出现预览图。
3. 点击预览图右上角 `×` 按钮。
4. 预期结果：预览图立即消失，`cover_image_url` 清空，可再次重新上传图片。

## E2E-002 详细文档显隐
1. 申报应用并上传详细文档，审批通过后进入首页应用详情。
2. 预期结果：详情页出现“详细文档”按钮并可打开文档。
3. 申报另一应用不上传详细文档并审批通过后进入详情。
4. 预期结果：不展示“详细文档”按钮。

## E2E-003 应用参与原子保存
1. 在“排行榜管理 -> 应用参与”打开“添加参与/编辑”弹窗。
2. 选择榜单并填写维度分值，点击保存。
3. 预期结果：一次保存成功，变更日志有记录，刷新后配置和维度分值同时生效。
4. 构造非法维度后提交。
5. 预期结果：返回字段级错误，配置和维度分值均不落库（无部分成功）。
